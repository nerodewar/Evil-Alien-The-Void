The Void v0.9.3 Hotfix

The title screen requires script.js. If GitHub contains script_v093.js instead of script.js, PLAY will not work.

Normal upload:
- index.html
- styles.css
- script.js

All three must be in the same folder and use these exact lowercase filenames.

Single-file emergency option:
- Rename index_single_file.html to index.html and upload it.
- This version contains the CSS and JavaScript internally, so no styles.css or script.js request is required.
- Keep your existing assets folder.
