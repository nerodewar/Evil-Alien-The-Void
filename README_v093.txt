The Void v0.9.3

Rendered deck-section map pass

What changed
- Rebuilt the map presentation around physical ship architecture rather than floating UI cards.
- Added bright metallic compartment hulls, inset dark deck wells, bevels, corner armour, ambient shadows and layered rim lighting.
- Converted routes into substantial corridor bodies with casing, floor channels, structural ribs, door frames and animated access pulses.
- Added room-specific top-down interiors: bunks, command consoles, Life Support machinery, laboratory benches, storage crates, mess furniture, security stations, tactical racks, airlocks, sat-nav hardware and reactor systems.
- Moved room labels inside each compartment and made the entire physical room the click / drag target.
- Added restrained deck telemetry and a small map legend without letting HUD text dominate the environment.
- Repositioned Luna inside rooms so her marker avoids the embedded labels more naturally.
- Improved current-room, objective, locked, complete, emergency and lockdown lighting states.
- Expanded the desktop map viewport and retained responsive mobile/tablet layouts.
- Preserved all current story content, checkpoints, room routes, interactions, branching logic, hiding outcome, cinematics and Luna token behaviour.
- Migrates v0.9.2 saves automatically into the v0.9.3 save slot.

Installation
Upload index.html, styles.css and script.js to the same GitHub directory as before. Keep your existing assets folder unchanged.
